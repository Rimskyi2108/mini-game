# 🎮 Сайт мини-игр

**Демо:** [Смотреть](https://rimskyi2108.github.io/mini-game/)

---

## 📌 Описание

Учебный проект — сайт с подборкой мини-игр на чистом JavaScript.

На главной странице представлен список игр, каждая из которых запускается по нажатию на кнопку.  
Также в репозитории есть папка `js/` с тренировочными файлами (hw-2.js – hw-9.js), в которых я выполнял различные задания по JavaScript.  
Файлы не связаны с основным проектом, но доступны для просмотра и проверки.

---

## 🛠 Технологии

- HTML5
- CSS3 (Flexbox, Grid, адаптив, анимации)
- JavaScript (DOM, события, логика игр, плавный скролл)

---

## 📱 Адаптивность

Проект адаптирован под устройства:
- Десктоп
- Мобильные телефоны

---

## ✨ Особенности

- 6 мини-игр на чистом JavaScript
- Анимированная полоса с названиями игр (якорные ссылки на игры)
- Адаптивная вёрстка
- Чистый, структурированный код
- Отдельная папка с тренировочными JS-заданиями

---

## 🎯 Список игр

- Угадай число
- Простая арифметика
- Переверни текст
- Камень, ножницы, бумага
- Простая викторина
